# Hagler Consulting Website

Simple landing page for GitHub Pages.

## Setup

1. Push this folder to a GitHub repo
2. Go to repo Settings â†’ Pages
3. Select branch `main` and folder `/website` (or `/docs`)
4. Your site will be live at `https://yourusername.github.io/repo-name`

## Custom Domain

1. Buy a domain (Namecheap, Cloudflare, etc.)
2. Rename `CNAME.example` to `CNAME` and add your domain
3. Add DNS records pointing to GitHub Pages

## Customization

Edit `index.html` to update content, colors, contact info.
